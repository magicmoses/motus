SPORTS — read all rules before assigning
════════════════════════════════════════

Allowed: running | cycling | rowing | skiing | hyrox | inline_skating | triathlon

RULE: Only assign a sport if the abstract's PRIMARY study population or PRIMARY application
is that sport. Do NOT assign based on physiological parameters alone.

Case 1 — Explicitly named single sport:
  "marathon runners", "road cyclists", "ergometer rowers", "cross-country skiers", etc.
  → Assign only that sport. Confidence 0.90+.

Case 2 — Named multi-sport or explicitly cross-sport population:
  "cyclists and runners", "endurance athletes from multiple disciplines",
  "triathletes", or study methods list multiple endurance sports.
  → Assign the explicitly named sports. Confidence 0.85.
  → "Endurance athletes" alone (without naming sports) → running + cycling only. Confidence 0.70.

Case 3 — Pure physiology, no sport population named:
  VO2max, lactate, HRV, cardiac output, sleep, nutrition, periodization
  studied in "trained athletes" or "recreationally active" without naming a sport.
  → Assign running + cycling only (most common endurance research populations). Confidence 0.70.
  → Exception — altitude papers: add skiing. Heat papers: running + cycling only.
  → DO NOT assign rowing or skiing unless the abstract mentions them.

Case 4 — Injury and biomechanics:
  Assign only the named sport. If no sport is named but injury is running-specific
  (achilles, patellofemoral, IT band, plantar fascia, shin splints) → running only. Confidence 0.70.
  Generic "athlete" injury papers without sport naming → do not assign any sport. Confidence 0.60.

Case 5 — Movement practice population (tai chi, yoga, martial arts, qi gong, pilates):
  These are NOT endurance sports. Do NOT assign sports[].
  Assign movement_practices[] instead.
  Exception: if the study explicitly tests a movement practice in endurance athletes
  (e.g. "yoga for marathon runner recovery") → assign both the sport AND movement_practice.

Case 6 — Sport mentioned only as comparator, context, or injury mechanism:
  "Compared to runners, cyclists showed..." → do NOT assign running if runners are not the study population.
  "Common in recreational athletes" → do NOT assign any sport.
  → Assign only the sport that IS the study population.

BEFORE writing "sports", write "_sport_reasoning" first. This forces you to verify each assignment.

OTHER FIELDS
══════════════════════════════════════

sample_size: integer only. First number that represents total participants. Null if unclear.

evidence_level:
  1 = RCT or meta-analysis
  2 = Cohort or controlled study
  3 = Case study or small N (< 20 participants)
  4 = Mechanistic or narrative review

Confidence:
  0.85–1.0 = clearly and explicitly stated
  0.60–0.84 = implied or inferred
  0.0–0.59  = uncertain or not mentioned
